# 🌦️ Weather Data Analysis Project

## 📌 Overview
This project analyzes weather data to understand patterns in **temperature, humidity, and precipitation** over time.  
The goal is to identify trends, relationships, and seasonal variations using Python.

---

## 🎯 Objective
- Analyze how weather variables change over time  
- Identify patterns and trends in temperature, humidity, and rainfall  
- Explore relationships between different weather factors  

---

## 📊 Dataset
The dataset contains:
- Date  
- Temperature (°C)  
- Humidity (%)  
- Precipitation  

---

## 🛠️ Tools & Technologies
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Google Colab  

---

## 🔍 Analysis Performed

### 1. Data Preprocessing
- Converted `Date` column to datetime format  
- Checked for missing values  
- Created new features like `Month`  

---

### 2. Exploratory Data Analysis (EDA)
- Distribution of temperature using histograms  
- Trend analysis using line plots  
- Correlation analysis using heatmap  

---

### 3. Monthly Analysis
- Calculated monthly averages for temperature, humidity, and precipitation  
- Identified seasonal trends  

---

## 📈 Key Insights
- Temperature shows significant variation (19°C – 46°C)  
- Humidity remains relatively stable across observations  
- Precipitation is irregular and occurs in spikes  
- Weak correlation exists between weather variables  
- Seasonal patterns are observed in monthly analysis  

---

## 🧾 Conclusion
The analysis reveals that temperature fluctuates significantly, while humidity remains relatively stable.  
Precipitation occurs irregularly, indicating uneven rainfall patterns.  

Monthly trends suggest seasonal variation, with certain months experiencing higher temperatures and rainfall.  

Overall, the dataset reflects a warm and humid climate with weak relationships between weather variables.
